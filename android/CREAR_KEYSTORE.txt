# INSTRUCCIONES PARA CREAR KEYSTORE
# ====================================
#
# 1. Ejecuta este comando en la carpeta android/:
#
#    keytool -genkey -v -keystore upload-keystore.jks -keyalg RSA -keysize 2048 -validity 10000 -alias upload
#
# 2. Te preguntará:
#    - Contraseña del keystore: (escribe una contraseña segura y GUÁRDALA)
#    - Contraseña de la key: (puede ser la misma que la anterior)
#    - Nombre y apellido
#    - Nombre de la organización: CargoClick
#    - Ciudad: Tu ciudad
#    - Estado: Tu región
#    - Código de país: CL (para Chile)
#
# 3. Confirma con "sí" al final
#
# 4. Crea el archivo key.properties con el siguiente contenido:
#
# storePassword=TU_CONTRASEÑA_AQUI
# keyPassword=TU_CONTRASEÑA_AQUI
# keyAlias=upload
# storeFile=upload-keystore.jks
#
# ⚠️ IMPORTANTE:
# - NUNCA subas estos archivos a Git (ya están en .gitignore)
# - Guarda el keystore en 3 lugares seguros (USB, nube privada, etc.)
# - Si pierdes el keystore NO podrás actualizar la app en Play Store
#
# 5. Una vez creado, este archivo ya no será necesario
